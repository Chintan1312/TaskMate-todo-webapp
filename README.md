### Run Locally

To run **TaskMate** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/ChintanKoradiya/TaskMate.git
```

Windows:

```bash
git clone https://github.com/ChintanKoradiya/TaskMate.git
```
